# 🍵 KansoChat

一个**可连发问题无需等待**的纯前端 AI 对话面板。单文件，零依赖，没有后端数据库，没有复杂的构建配置。KansoChat 的目标是：提供一个流畅、顺手、但又克制的文字交互环境。

>🍃 注：本项目极度克制（我已经禁食145天！），目前仅适配并支持 Google Gemini API。**

## ✨ 核心特性

### 🚀 连发机制 

* **思考不打断**：当 AI 正在流式生成上一条回复时，你完全可以继续输入新的想法并发送。
* **平滑列队**：请求会被自动加入并发队列（可在设置中自定义队列容量），按顺序静默处理，让对话节奏始终跟上你的心流。

### 🍵 简素（Kanso）即正义

* **单文件运行**：克隆或下载 `index.html`，双击在浏览器中打开即可使用。
* **数据隐私**：所有对话历史、配置选项、API Key 均使用 Local Storage 加密存储于你的本地浏览器，纯粹的本地化体验。

### 🎨 克制的设计与交互

* **无缝悬浮 UI**：可选开启底部背景平滑渐变透明，最大程度释放屏幕纵向阅读空间，让文字如流水般隐入。
* **精准的上下文控制**：精美的全局滑动条，可直观控制每次提问携带的历史记录条数（0 ~ ∞）。
* **代码块智能处理**：超过 12 行的长代码块默认自动折叠中间部分，并支持单击区域极速复制。
* **透明度预估**：在“思考中”状态下，微弱显示本次请求预估消耗的 Token 数量，心中有数。

### ⌨️ 高效快捷键流 

专为高频使用者打磨，双手无需离开键盘：

* <kbd>Cmd/Ctrl</kbd> + <kbd>Enter</kbd> : 发送消息 / 将消息加入排队
* <kbd>Alt</kbd> + <kbd>N</kbd> : 快速新建空对话
* <kbd>Alt</kbd> + <kbd>F</kbd> : 切换沉浸式全屏输入框
* <kbd>Alt</kbd> + <kbd>↑</kbd> / <kbd>↓</kbd> : 纵向无缝切换历史对话
* <kbd>Cmd/Ctrl</kbd> + <kbd>K</kbd> : 插入/撤销上下文清除线（物理截断 AI 记忆）
* <kbd>Alt</kbd> + <kbd>1~9</kbd> : 极速切换自定义的专属模型

## 🌟 快速上手

1. 下载本仓库中的 `KansoChat.html`。
2. 使用现代浏览器（推荐 Chrome / Edge / Safari）打开。
3. 点击左下角 ⚙️ **偏好设置** -> **系统**，填入你的 **Google Gemini API Key**。
4. （可选）在 **模型** 选项卡中，添加你常用的 Gemini 模型（如 `gemini-1.5-pro`），并切换全局默认模型。
5. 开始沉浸式对话。
